Blinkit Sales Analysis (Retail Data Analytics)
Files: Blinkit Dataset.xlsx, Report.docx

Project Overview: An end-to-end analysis of quick-commerce data to identify delivery performance, product popularity, and revenue trends.

Key Features:

  Data Cleaning: Processed raw transactional data for consistency.

  Analysis: Analyzed peak ordering times and high-performing categories.

  Reporting: Documented actionable insights to optimize inventory and delivery logistics.
